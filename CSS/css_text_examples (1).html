CSS 텍스트 속성 예제 모음
예제 01
font-size / color (#2c3e50)
글자 크기를 22px로 키우고, 짙은 남색 계열의 16진수 색상을 지정합니다. font-weight: 700으로 굵게 강조합니다.
텍스트 크기와 색상
font-size:22px; color:#2c3e50; font-weight:700;
예제 02
font-style / letter-spacing (#8e44ad)
italic으로 기울이고 letter-spacing:4px로 자간을 넓혀 보라색 계열 색상과 함께 우아한 느낌을 줍니다.
이탤릭 자간 예제
font-style:italic; letter-spacing:4px; color:#8e44ad;
예제 03
text-shadow (#e74c3c)
X·Y 오프셋과 블러 반경을 이용해 붉은 계열 그림자를 두 겹으로 적용, 입체적인 발광 효과를 만듭니다.
텍스트 그림자 효과
color:#e74c3c; text-shadow:2px 2px 4px #f1948a, -1px -1px 3px #fadbd8;
예제 04
text-decoration (#16a085)
underline overline을 동시에 적용하고 wavy 스타일로 물결 밑줄·윗줄을 청록색으로 표시합니다.
밑줄 + 윗줄 웨이브
color:#16a085; text-decoration:underline overline wavy #1abc9c;
예제 05
text-transform (#d35400)
uppercase로 모든 글자를 대문자로 변환하고, 주황색 계열 색상과 letter-spacing으로 타이틀 느낌을 강조합니다.
uppercase transform
text-transform:uppercase; letter-spacing:2px; color:#d35400;

예제 06
direction / unicode-bidi (#1a5276)
direction:rtl로 텍스트 흐름을 오른쪽→왼쪽으로 설정합니다. 아랍어·히브리어 등 RTL 언어 레이아웃에 활용됩니다.
RTL 방향 텍스트 ◀
direction:rtl; unicode-bidi:embed; color:#1a5276;
예제 07
text-shadow 네온 글로우 (#f1c40f)
어두운 배경 위에 노란색 글자와 두 단계의 text-shadow로 네온사인 발광 효과를 구현합니다.
NEON GLOW
color:#f1c40f; text-shadow:0 0 6px #f9e45a, 0 0 14px #f39c12;
예제 08
word-spacing / text-align (#117a65)
word-spacing:10px으로 단어 간격을 넓히고 justify로 양쪽 정렬해 균형 잡힌 단락을 만듭니다.
단어 사이 간격을 넓혀 정렬한 예시 문장입니다.
word-spacing:10px; text-align:justify; color:#117a65;
예제 09
line-height / font-weight (#6c3483)
line-height:2.2으로 행간을 넉넉하게 확보하고 font-weight:600으로 준굵게 처리해 가독성을 높입니다.
첫 번째 줄 텍스트
두 번째 줄 텍스트
line-height:2.2; font-weight:600; color:#6c3483;
예제 10
다중 text-shadow 입체 효과 (#1b2631)
네 개의 text-shadow를 1px씩 누적해 마치 블록 글자처럼 보이는 3D 입체 효과를 만듭니다. 가장 마지막 레이어에 블러를 추가해 깊이감을 강조합니다.
3D Shadow Text Effect
color:#1b2631;
text-shadow: 1px 1px 0 #aab7b8, 2px 2px 0 #7f8c8d, 3px 3px 0 #566573, 4px 4px 6px rgba(0,0,0,.3);

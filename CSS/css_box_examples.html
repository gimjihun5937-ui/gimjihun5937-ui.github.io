CSS 경계선(Border) 종류와 예제
① border-style — 선 스타일 8종
solid
가장 기본적인 실선. 버튼·카드 등 범용으로 사용됩니다.
solid
border: 3px solid #3b5bdb;
dashed
파선(대시). 구분선, 드래그 영역, 점선 박스 등에 활용됩니다.
dashed
border: 3px dashed #e03131;
dotted
점선. 아이콘 강조나 링크 밑줄 대체에 자주 사용됩니다.
dotted
border: 3px dotted #2f9e44;
double
이중선. 두께가 충분할 때 두 줄로 표현됩니다.
double
border: 6px double #f08c00;
groove
마치 파인 홈처럼 입체적으로 보이는 3D 효과 선입니다.
groove
border: 5px groove #9c36b5;
ridge / inset / outset
ridge: 돌출 입체선  |  inset: 눌린 효과  |  outset: 볼록 효과
ridge
inset
outset
border: 5px ridge #1098ad;
border: 5px inset #c2255c;
border: 5px outset #5c7cfa;
② 개별 방향 border (top / right / bottom / left)
네 방향을 각각 다른 색·굵기·스타일로 지정할 수 있습니다.
border-top
border-bottom
border-top: 4px solid #3b5bdb;
border-bottom: 4px solid #2f9e44;
좌·우 단독 적용 예시와 4방향 각각 다른 색 적용 예시입니다.
border-left
4방향 다른 색
border-left: 4px solid #f08c00;
border-color: #3b5bdb #e03131 #2f9e44 #f08c00;
③ border-radius — 모서리 둥글기
소·중·알약·원형
값이 클수록 더 둥글어지며, 50%이면 완전한 원이 됩니다.
4px
12px
50px
50%
border-radius: 4px / 12px / 50px / 50%;
none / hidden
none은 선 없음, hidden은 공간은 차지하지만 보이지 않습니다.
border: none
border: hidden (공간 유지)
border: none;
border: 3px hidden #aaa;

CSS Margin · Padding 종류와 예제
① 박스 모델 구조 다이어그램
MARGIN (바깥 여백)
BORDER (경계선)
PADDING (안쪽 여백)
CONTENT (콘텐츠 영역)
② margin — 요소 바깥 여백 9종
margin (전체)
4방향 동일 여백. 단축 속성으로 한 번에 지정합니다.
margin:16px
margin: 16px;
margin-top / bottom
상·하 방향에만 바깥 여백을 적용합니다.
상하 16px
margin-top:16px;
margin-bottom:16px;
margin-left / right
좌·우 방향에만 바깥 여백을 적용합니다.
좌우 20px
margin-left:20px;
margin-right:20px;
margin-top 단독
위쪽 여백만 따로 지정합니다.
top only
margin-top: 16px;
margin-bottom 단독
아래쪽 여백만 따로 지정합니다.
bottom only
margin-bottom: 16px;
margin: 0 auto
좌우를 auto로 설정해 수평 중앙 정렬을 만듭니다.
가운데
margin: 0 auto; width:70%;
③ padding — 요소 안쪽 여백 9종
padding (전체)
4방향 동일 안쪽 여백. 콘텐츠와 경계선 사이 공간입니다.
padding:20px
padding: 20px;
padding 상하 / 좌우
단축 2값: 첫째=상하, 둘째=좌우 여백입니다.
8px 20px
padding: 8px 20px;
padding 4값 단축
top→right→bottom→left 시계 방향으로 각각 지정합니다.
4 12 8 20px
padding: 4px 12px 8px 20px;
padding-top
위쪽 안쪽 여백만 지정합니다.
top:20px
padding-top: 20px;
padding-right
오른쪽 안쪽 여백만 지정합니다.
right:24px
padding-right: 24px;
padding-bottom / left
아래·왼쪽 안쪽 여백을 각각 단독으로 지정합니다.
bottom:20px
left:24px
padding-bottom:20px;
padding-left:24px;
④ margin vs padding 비교 요약
구분 위치 배경색 적용 음수 사용 주요 용도
margin 요소 바깥 ❌ 투명 ✅ 가능 요소 간 간격, 중앙 정렬
padding 요소 안쪽 ✅ 적용됨 ❌ 불가 콘텐츠 여백, 클릭 영역 확장

CSS 박스(Box) & 그림자(Shadow) 예제
① box-shadow 기본 구문
box-shadow: offset-x offset-y blur-radius spread-radius color [inset];
  — offset-x/y: 그림자 위치  |  blur: 흐림 정도  |  spread: 크기  |  inset: 내부 그림자
② box-shadow 예제 10종
기본 그림자
X·Y 4px 오프셋에 8px 블러. 가장 일반적인 카드 그림자입니다.
기본 box-shadow
box-shadow: 4px 4px 8px rgba(0,0,0,.2);
spread (확산) 그림자
네 번째 값 spread로 그림자를 사방으로 넓게 퍼뜨립니다.
spread 4px
box-shadow: 0 4px 16px 4px rgba(59,91,219,.35);
inset (내부) 그림자
inset 키워드로 그림자가 요소 안쪽에 그려져 눌린 효과를 냅니다.
inset shadow
box-shadow: inset 0 4px 10px rgba(0,0,0,.2);
다중 그림자 (multi)
쉼표로 구분해 여러 그림자를 레이어처럼 겹칩니다.
다중 그림자
box-shadow: 4px 4px 0 #3b5bdb, 8px 8px 0 #e03131;
네온 글로우 (neon)
blur만 크게 주고 색상에 채도를 높여 네온 발광 효과를 만듭니다.
NEON GLOW
box-shadow: 0 0 8px #3b5bdb, 0 0 20px #5c7cfa, 0 0 40px #748ffc;
위쪽 그림자 (top)
Y 오프셋을 음수로 설정해 그림자가 요소 위쪽에 생깁니다.
위쪽 그림자
box-shadow: 0 -6px 12px rgba(59,91,219,.3);
카드 UI 그림자
가까운 그림자(작은 블러)와 먼 그림자(큰 블러)를 합쳐 자연스러운 카드 느낌을 만듭니다.
Card Shadow
box-shadow: 0 2px 4px rgba(0,0,0,.08),
            0 8px 24px rgba(0,0,0,.12);
하드 그림자 (브루탈리즘)
blur 없이 오프셋만 주어 딱딱하고 각진 복고풍 그림자를 만듭니다.
Hard Shadow
box-shadow: 6px 6px 0 #1e1e2e;
border: 2px solid #1e1e2e;
소프트 부유 그림자
offset 0, 큰 blur·spread로 마치 공중에 떠 있는 듯한 부드러운 그림자를 표현합니다.
Floating
box-shadow: 0 20px 40px rgba(0,0,0,.08);
border-radius: 16px;
컬러 그림자
그림자 색상에 배경과 같은 계열의 색을 주어 요소가 빛나는 느낌을 연출합니다.
Colored Shadow
box-shadow: 0 8px 20px rgba(224,49,49,.45);
background: #fff5f5;
③ box-shadow 값 요약
값 의미 기본값
offset-x가로 방향 이동 (+오른쪽 / -왼쪽)0
offset-y세로 방향 이동 (+아래 / -위)0
blur-radius흐림 정도 (클수록 부드럽게)0
spread-radius그림자 크기 확대/축소0
color그림자 색상 (rgba 권장)currentColor
inset키워드 추가 시 내부 그림자로 전환없음(외부)
